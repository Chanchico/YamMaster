# YamMaster

## Installation des dépendances
Dans la racine du projet
```
npm install
```

Dans /websocket-game-server
```
npm install
```

## Exécution du client :
Dans la racine du projet
```
 npx expo start      
```
## Exécution du serveur
Dans /websocket-game-server
```
 npm run start
```
